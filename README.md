📊 E-Commerce Sales Analysis Dashboard (Power BI)

This project presents an interactive Power BI dashboard created to analyze e-commerce sales data and generate meaningful business insights. The dashboard helps visualize sales performance, profit trends, customer behavior, and payment methods using dynamic charts and filters.

📁 Project Overview

The dataset contains information related to orders, sales amount, profit, quantity, product categories, payment modes, customer details, and locations. Using Power BI, the data was transformed and visualized to better understand overall business performance.

📊 Dashboard Features

The dashboard includes the following key insights:

Total Sales Amount: 438K

Total Profit: 37K

Total Quantity Sold: 5615

Average Order Value (AOV): 121K

Visualizations Included

Amount by State – Shows sales contribution from different states.

Quantity by Payment Mode – Distribution of orders by payment methods such as COD, UPI, Debit Card, Credit Card, and EMI.

Profit by Sub-Category – Profit comparison across different product sub-categories.

Orders by City Map – Geographic visualization of orders across cities in India.

Amount by Customer Name – Top customers contributing to total sales.

Quantity by Category – Category-wise sales distribution.

Profit by Month – Monthly profit trends throughout the year.

Interactive filters for Quarter and State allow users to dynamically explore the data.
